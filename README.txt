This repository contains extra software for Slackware Linux, suitable for
desktops and servers. 

Complete sources, build scripts and step-by-step documentation is included. 

Binary packages are provided for 32-bit and 64-bit architectures.

You'll also find a collection of Linux-HOWTOs (in french).

Check out the README and ChangeLog.txt files in the respective directory trees
for more information.
